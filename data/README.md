# Ammunition Trade Mobilization and NATO Europe’s Wartime Resilience

This repository contains data and replication materials for the manuscript:

**“Beyond Defence Spending: Ammunition Trade Mobilization and the Limits of NATO Europe’s Wartime Resilience.”**

## Repository structure

```text
data/
  raw/          Original source files used for data construction
  processed/   Cleaned or analysis-ready datasets, if added later

scripts/       Replication scripts, if added later

outputs/
  tables/      Replication tables, if added later
  figures/     Replication figures, if added later

MANIFEST.csv   File inventory and checksum information
```

## Raw data sources included

The `data/raw/` folder contains source files related to:

- NATO defence expenditure data
- SIPRI military expenditure data
- UN Comtrade ammunition trade records
- World Bank manufacturing value added data
- An internal data collection plan/codebook file for the ammunition resilience study

## Data availability statement

The data underlying this study are derived from publicly available sources, including NATO defence expenditure data, SIPRI military expenditure data, UN Comtrade ammunition trade records, and World Bank manufacturing indicators. The raw source files used for data construction are included in this repository. Processed datasets and replication scripts may be added as the replication package is finalized.
