# Data

This folder contains the source files used to construct the dataset for the ammunition trade mobilization study.

## Folders

- `raw/`: original source files
- `processed/`: cleaned or analysis-ready datasets, if added later

## Main source categories

- NATO defence expenditure data
- SIPRI military expenditure data
- UN Comtrade ammunition trade data
- World Bank manufacturing value added data
- Data collection/codebook material for the study

See `MANIFEST.csv` in the repository root for file names, original file names, sizes, and checksums.
